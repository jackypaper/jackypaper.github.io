Hugo It

Firstly, need to install hugo correctly.

Initialization

Then, start to build your website.

$ hugo new site jackypaper
$ cd jackypaper
$ tree -a

Add Theme

I choose hemingway2 as my page theme.

$ cd theme
$ git clone https://github.com/beli3ver/hemingway2.git

Update config.toml

baseurl = "https://example.com"
languageCode = "en"
title = "Hemingway2"
theme = "hemingway2"

Start the Server

$ cd ..
$ hugo server --theme=hemingway2 --buildDrafts

Make posts public

$ hugo undraft content/post/good-to-great.md
$ hugo server --theme=hemingway2

Generate website

% In config.coml
baseURL = "https://<your GitHub username>.github.io/jackypaper/"
[or]
baseURL = "https://jackypaper.github.io/"

$ hugo --theme=hemingway2

Deploy bookshelf on GitHub pages

# root folder
$ git init
$ echo "/public/" >> .gitignore
$ echo "/themes/" >> .gitignore
$ git add --all
$ git commit -m "Initial commit"

$ cd public

# If necessary, Please check:
$ git config user.name
$ git config user.name jackypaper

$ git init
$ git remote add origin https://github.com/jackypaper/jackypaper.github.io.git

If necessary, for subdomain

$ git checkout -b gh-pages
Switched to a new branch 'gh-pages'

Add all the files (within jackypaper/public) to the index, commit them, and push the changes to GitHub.

$ git add --all
$ git commit -m "new commit"
$ git push -f origin master
[or, for subdomin]
$ git push -f origin 'gh-pages'

Regenerate page

$ (cd ..; hugo --theme=hemingway2)

$ cd public
$ git add --all
$ git commit -m "<some change message>"
$ git push -f origin master <or 'gh-pages'>
